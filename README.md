# DHCP Сервер на Node.js 
Простой DHCP сервер, реализованный на Node.js.

## Требования 
- Node.js 14 или выше
## Установка 

```bash
npm install
```

## Запуск
```bach
node server.js
``` 

## Конфигурация
По умолчанию сервер настроен со следующими параметрами:
- Интерфейс: eth0
- Диапазон IP: 192.168.1.100
- 192.168.1.200
- Маска подсети: 255.255.255.0
- Шлюз: 192.168.1.1
- DNS: 8.8.8.8, 8.8.4.4
- Время аренды: 24 часа
